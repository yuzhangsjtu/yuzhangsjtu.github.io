### Common Mac Keyboard Shortcuts

This section is excerpted from Apple's official guide: [Mac keyboard shortcuts](https://support.apple.com/en-us/HT201236)

To use a keyboard shortcut, hold down one or more modifier keys while pressing the last key of the shortcut. For example, to use Command-C (Copy), hold down the Command key, press C, then release both keys at the same time. Mac menus and keyboards often [use symbols for certain keys](https://support.apple.com/guide/mac-help/what-are-those-symbols-shown-in-menus-cpmh0011/mac), including the following modifier keys:

- Command (or Cmd) ⌘
- Shift ⇧
- Option (or Alt) ⌥
- Control (or Ctrl) ⌃
- Caps Lock ⇪
- Fn 🌐

On keyboards designed for Windows PCs, use the Alt key instead of Option, and the Windows logo key instead of Command.
