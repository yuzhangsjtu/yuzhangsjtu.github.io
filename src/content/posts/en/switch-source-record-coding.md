Yesterday I checked out a page that a member of the Fluid chat group had deployed on Coding, and found the access speed to be excellent. So today I'm tinkering with it myself to give it a try!

Deploying on Coding is fairly simple — you can clone directly from a GitHub repository. But afterwards, setting up dual Git deployment to both GitHub and Coding took a lot of time. It's always like this: as a newbie taking a swing, something others can finish in three minutes takes me a whole day of fiddling.

In the afternoon, I pointed [yuzhang.xyz](https://yuzhang.xyz) to Coding as well, and it took effect immediately.

But [yuzhangnju.me](https://yuzhangnju.me), resolved through Cloudflare, still hasn't taken effect — maybe I got the settings wrong... anyway, whatever happens happens, it's an orphan blog regardless...
