Yesterday, I observed a Fluid community member's page deployed on Coding, and the access speed was remarkably fast. So today I decided to tinker with it myself!

Deploying on Coding was relatively straightforward — just clone directly from a GitHub repository. But afterwards, setting up dual Git deployment to both GitHub and Coding took up a lot of time. Always the way: when a rookie takes a swing, something that takes others three minutes costs me a whole day.

In the afternoon, I pointed [yuzhang.xyz](https://yuzhang.xyz) to Coding as well, and it took effect immediately.

But [yuzhangnju.me](https://yuzhangnju.me), resolved through Cloudflare, still hasn't kicked in — maybe I messed up the settings… anyway, let it be. It's just an orphan blog, after all…

Last night, logging into Cloudflare, I found overseas traffic and realized: my little orphan blog can actually be found on Google (Baidu has never indexed it). So I turned on visitor tracking (Busuanzi stats seem to have errors), haha~

Hello, stranger.
