### [Keyboard Shortcuts in Terminal on Mac](https://support.apple.com/zh-cn/guide/terminal/trmlshtcts/2.13/mac/13.0)

In the Terminal app 🧑🏻‍💻 on your Mac, you can use keyboard shortcuts to quickly accomplish many tasks. Refer to the shortcuts below, as well as the shortcuts shown in the Terminal menu in the [menu bar](https://support.apple.com/zh-cn/guide/terminal/aside/glos33eb8abd/2.13/mac/13.0). In the app menus, keyboard shortcuts are [represented by symbols](https://support.apple.com/zh-cn/guide/mac-help/cpmh0011).

**Note:** Keyboard shortcuts in an app vary depending on the language and keyboard layout you use on your Mac. If the shortcuts below don't work as expected, look in the app menus in the menu bar to find the correct shortcuts. You can also [use the Keyboard Viewer](https://support.apple.com/zh-cn/guide/mac-help/mchlp1015) to see your current keyboard layout, known as an *input source*.

#### Use Terminal windows and tabs

| Action                              | Shortcut                       |
| :---------------------------------- | :----------------------------- |
| New window                          | Command-N                      |
| New window with the same command    | Control-Command-N              |
| New tab                             | Command-T                      |
| New tab with the same command       | Control-Command-T              |
| Show or hide the tab bar            | Shift-Command-T                |
| Show all tabs or exit tab overview  | Shift-Command-Backslash (\)    |
| New command                         | Shift-Command-N                |
| New remote connection               | Shift-Command-K                |
| Show or hide the inspector          | Command-I                      |
| Edit title                          | Shift-Command-I                |
| Edit background color               | Option-Command-I               |
| Make the font bigger                | Command-Plus Sign (+)          |
| Make the font smaller               | Command-Minus Sign (–)         |
| Next window                         | Command-Grave Accent (`)       |
| Previous window                     | Command-Shift-Tilde (~)        |
| Next tab                            | Control-Tab                    |
| Previous tab                        | Control-Shift-Tab              |
| Split the window into two panes     | Command-D                      |
| Close the split pane                | Shift-Command-D                |
| Close tab                           | Command-W                      |
| Close window                        | Shift-Command-W                |
| Close other tabs                    | Option-Command-W               |
| Close all                           | Option-Shift-Command-W         |
| Scroll to top                       | Command-Home                   |
| Scroll to bottom                    | Command-End                    |
| Page up                             | Command-Page Up                |
| Page down                           | Command-Page Down              |
| Line up                             | Option-Command-Page Up         |
| Line down                           | Option-Command-Page Down       |

#### Edit a command line

| Action                                      | Shortcut                                                       |
| :------------------------------------------ | :------------------------------------------------------------ |
| Reposition the insertion point              | While holding down the Option key, move the pointer to the new insertion point |
| Move the insertion point to the beginning of the line | Control-A                                            |
| Move the insertion point to the end of the line | Control-E                                                |
| Move the insertion point forward one character | Right Arrow                                               |
| Move the insertion point back one character | Left Arrow                                                    |
| Move the insertion point forward one word   | Option-Right Arrow                                            |
| Move the insertion point back one word      | Option-Left Arrow                                             |
| Delete the line                             | Control-U                                                     |
| Delete to the end of the line               | Control-K                                                     |
| Delete forward to the end of the word       | Option-D (available after selecting [Use Option as Meta key](https://support.apple.com/zh-cn/guide/terminal/trmlkbrd/2.13/mac/13.0)) |
| Delete backward to the beginning of the word | Control-W                                                    |
| Delete one character                        | Delete                                                        |
| Delete one character forward                | Forward Delete (or use Fn-Delete)                            |
| Transpose two characters                    | Control-T                                                     |
