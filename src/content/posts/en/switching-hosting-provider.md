Earlier, on the recommendation of a senior lab mate, I got half a year of Alibaba Cloud ECS. But my domain name isn't ICP-filed, so I probably can't use it. Besides, Alibaba Cloud will eventually start charging, so finding a stable repository to store my blog became a hard necessity.

I briefly tried out [Gitee](https://gitee.com/): it can seamlessly fork GitHub repos, but it's very slow and very unstable. After some searching, I learned about the Vercel platform (formerly named Zeit). Their servers are said to be in Taiwan; I gave it a try, and it was very fast, with the lowest latency being only 50ms.

Vercel has many advantages:

- Seamlessly links to a GitHub repo. After deploying locally, Vercel can sync instantly.
- The settings pages are clean and user-friendly.
- It's free.

So, this blog is temporarily hosted on Vercel's servers, while at the same time the GitHub Page is kept. Both the github.io site and the Vercel site point to [yuzhangnju.me](https://yuzhangnju.me/).
